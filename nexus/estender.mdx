## Endpoint

<CodeGroup>

```bash Requisição
POST /nexus/extend/{id}
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Estende a validade de um usuário Nexus existente.

## Parâmetros

<ParamField header="Authorization" type="string" required>
  Bearer token para autenticação
</ParamField>

<ParamField path="id" type="string" required>
  ID do usuário Nexus
</ParamField>

<ParamField query="credits" type="number" required>
  Créditos para extensão
</ParamField>

<ParamField query="screens" type="integer" required>
  Número de telas
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X POST "https://api-new.paineloffice.click/nexus/extend/{id}?credits=0.5&screens=1" \
  -H "Authorization: Bearer <token>"
```

</CodeGroup>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      ID do usuário Nexus
    </ResponseField>
    <ResponseField name="username" type="string">
      Nome de usuário
    </ResponseField>
    <ResponseField name="previous_expires_at" type="string">
      Data de expiração anterior
    </ResponseField>
    <ResponseField name="new_expires_at" type="string">
      Nova data de expiração
    </ResponseField>
    <ResponseField name="days_extended" type="integer">
      Dias estendidos
    </ResponseField>
    <ResponseField name="bandwidth_limit" type="integer">
      Novo limite de banda
    </ResponseField>
  </Expandable>
</ResponseField>
