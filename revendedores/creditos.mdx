---
title: "Gerenciar Créditos"
api: "PATCH https://api.officehub.top/resale/add-credits/{id}"

auth: "none"
---

## Endpoint

<CodeGroup>

```bash Requisição
PATCH /resale/add-credits/{id}
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Adiciona ou remove créditos de uma revenda específica. Use valores positivos para adicionar créditos e valores negativos para remover.

## Parâmetros

<ParamField header="Authorization" type="string" required>
  Bearer token para autenticação
</ParamField>

<ParamField path="id" type="integer" required>
  ID da revenda
</ParamField>

## Parâmetros do Body

<ParamField body="credits" type="string" required>
  Quantidade de créditos a adicionar ou remover (Mínimo 10 créditos por operação)
</ParamField>

<ParamField body="reason" type="string">
  Motivo da alteração de créditos
</ParamField>

<ParamField body="sale" type="number">
  Valor de venda associado
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      ID da revenda
    </ResponseField>
    <ResponseField name="previous_credits" type="number">
      Quantidade de créditos antes da operação
    </ResponseField>
    <ResponseField name="new_credits" type="number">
      Quantidade de créditos após a operação
    </ResponseField>
    <ResponseField name="credits_added" type="number">
      Quantidade de créditos adicionados/removidos
    </ResponseField>
    <ResponseField name="reason" type="string">
      Motivo da alteração
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Requisição

<CodeGroup>

```json Adicionar Créditos
{
  "credits": "20",
  "reason": "Recarga manual",
  "sale": 100.02
}
```

```json Remover Créditos
{
  "credits": "-15",
  "reason": "Ajuste de saldo",
  "sale": 0
}
```

```bash cURL (Adicionar)
curl -X PATCH "https://api.officehub.top/resale/add-credits/15" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "credits": "20",
    "reason": "Recarga manual",
    "sale": 100.02
  }'
```

```bash cURL (Remover)
curl -X PATCH "https://api.officehub.top/resale/add-credits/15" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "credits": "-15",
    "reason": "Ajuste de saldo",
    "sale": 0
  }'
```

</CodeGroup>

## Exemplo de Resposta

**Sucesso (200 OK) - Adicionando Créditos:**
```json
{
  "success": true,
  "message": "Créditos atualizados com sucesso",
  "data": {
    "id": 15,
    "previous_credits": 100,
    "new_credits": 120,
    "credits_added": 20,
    "reason": "Recarga manual"
  }
}
```

**Sucesso (200 OK) - Removendo Créditos:**
```json
{
  "success": true,
  "message": "Créditos atualizados com sucesso",
  "data": {
    "id": 15,
    "previous_credits": 120,
    "new_credits": 105,
    "credits_added": -15,
    "reason": "Ajuste de saldo"
  }
}
```

**Erro (404 Not Found):**
```json
{
  "success": false,
  "message": "Revenda não encontrada",
  "error": "RESALE_NOT_FOUND"
}
```

**Erro (400 Bad Request):**
```json
{
  "success": false,
  "message": "Quantidade mínima de créditos é 10",
  "error": "INVALID_CREDITS_AMOUNT"
}
```
