---
title: "Criar Revenda"
api: "POST https://api.officehub.top/resale"
description: "Cria uma nova revenda no sistema"
auth: "none"
---

## Endpoint

<CodeGroup>

```bash Requisição
POST /resale
Authorization: Bearer <token>
```

</CodeGroup>

## Descrição

Cria uma nova revenda no sistema com os dados fornecidos. A revenda criada terá acesso ao painel para gerenciar seus próprios clientes.

## Parâmetros

<ParamField header="Authorization" type="string" required>
  Bearer token para autenticação
</ParamField>

<ParamField body="user" type="string" required>
  Nome de usuário da revenda
</ParamField>

<ParamField body="pass" type="string" required>
  Senha da revenda
</ParamField>

<ParamField body="credits" type="number" required>
  Créditos iniciais (Mínimo 10 créditos)
</ParamField>

<ParamField body="email" type="string">
  E-mail da revenda
</ParamField>

<ParamField body="whatsapp" type="number">
  WhatsApp da revenda (apenas números, sem formatação)
</ParamField>

<ParamField body="notes" type="string">
  Notas e observações sobre a revenda
</ParamField>

<ParamField body="master" type="boolean" default="false">
  Define se a revenda é master (true/false)
</ParamField>

<ParamField body="pattern" type="integer" default="0">
  Padrão do sistema
</ParamField>

<ParamField body="sale" type="number">
  Valor de venda
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem de confirmação
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      ID da revenda criada
    </ResponseField>
    <ResponseField name="username" type="string">
      Nome de usuário da revenda
    </ResponseField>
    <ResponseField name="email" type="string">
      Email da revenda
    </ResponseField>
    <ResponseField name="credits" type="number">
      Créditos da revenda
    </ResponseField>
    <ResponseField name="status" type="string">
      Status da revenda
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Data de criação
    </ResponseField>
  </Expandable>
</ResponseField>

## Exemplo de Requisição

<CodeGroup>

```json TryIt
{
  "user": "revenda_teste",
  "pass": "senha123",
  "credits": 10,
  "email": "teste@gmail.com",
  "whatsapp": 5511999999999,
  "notes": "Revenda de teste",
  "master": false,
  "pattern": 0,
  "sale": 10
}
```

```bash cURL
curl -X POST "https://api.officehub.top/resale" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "user": "revenda_teste",
    "pass": "senha123",
    "credits": 10,
    "email": "teste@gmail.com",
    "whatsapp": 5511999999999,
    "notes": "Revenda de teste",
    "master": false,
    "pattern": 0,
    "sale": 10
  }'
```

</CodeGroup>

## Exemplo de Resposta

**Sucesso (201 Created):**
```json
{
  "success": true,
  "message": "Revenda criada com sucesso",
  "data": {
    "id": 15,
    "username": "revenda_teste",
    "email": "teste@gmail.com",
    "credits": 10,
    "status": "active",
    "created_at": "2024-01-15T10:30:00Z"
  }
}
```

**Erro (400 Bad Request):**
```json
{
  "success": false,
  "message": "Créditos insuficientes. Mínimo 10 créditos",
  "error": "INVALID_CREDITS"
}
```

**Erro (422 Unprocessable Entity):**
```json
{
  "success": false,
  "message": "Dados de entrada inválidos",
  "errors": {
    "user": ["O campo user é obrigatório"],
    "pass": ["O campo pass é obrigatório"],
    "credits": ["O campo credits deve ser no mínimo 10"]
  }
}
```
